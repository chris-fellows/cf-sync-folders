CFSyncFolders

Synchronizes a number of source folders to a corresponding destination folder. This allows you to configure particular Windows folders to be automatically backed up to an external hard drive. For example, Documents, Music, Pictures, Videos, other custom folders.

Supports the number of destination types:
  • Local file system, removable drive or network drive.
  • Dropbox (In progress)
  • Microsoft One Drive (In progress)
  • Google Drive (In progress)
Features:
  • Destination folder path supports placeholders that can be set at runtime. Supported placeholders include current year number, month number, day number, user name, machine name. The date placeholders would be used where it was necessary to have separate backups based on the date. For example, If you wanted to retain separate backups for previous months then you would specify the month number placeholder in the destination folder path.
  • Destination folder path supports the ability to specify the volume label for the destination drive instead of indicating a specific driver letter. It allows the system to back up to removable drives that may be assigned a different drive letter each time that they are connected.
The application can be run in any of the following ways:
  1. Interactively. The application will only synchronize when the user requests it.
  2. In the background from the Task Tray (bottom right near the clock) and periodically synchronize.
  3. Silently from the command line. For example, From the Windows Scheduler.
